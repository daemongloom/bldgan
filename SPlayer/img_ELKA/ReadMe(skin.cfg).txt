Редактирование skin.cfg

Открываем skin.cfg любым HEX редактором и далее:

Координата сумма байтов(ибо FF возвращает 256 значений а на ELKA максимальное-320)

Дабы сэкономить память телефона :) сделаем немного по-другому! :)
Для координат X выделяем 1 байт, для Y - 2. Сэкономили драгоценные 5 байт! :)

Громкость
x  0x02
y  0x03+0x04
Статус
x  0x05
y  0x06+0x07
Список треков(началом является 3й трек)
x  0x08
y  0x09+0x0A
Интервал между треками
0xB
Нумерация (играющий/выбранный/общее количество)
x  0x0C
y  ox0D+0x0E
Иконка random.png
x  0x0F
y  0x10+0x11
Иконка блок.клав.
x  0x12
y  0x13+0x14
